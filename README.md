📖 About the project

AlphaZ Browser is a full—fledged web browser based on [Electron](https://electronjs.org /). The project was created from scratch as an alternative to existing browsers, with an emphasis on simplicity, speed and security.

⚠️ ATTENTION: The project is at an early stage of development. Some functions may be unstable or missing. I'm just starting my career in browser development!



🚀 Features (under development)

 ✅ It's already working

- 🚀 Quick Start — starts in a split second
- 📑 Tabs — as in normal browsers
- 🔍 Smart Search — input hints (Yandex)
- 🎨 **Design themes — light, dark (classic/modern), its own color
- 🔧 Settings — convenient panel on the right
- 🖥️ Full—screen mode - by pressing F11
- 📦 Session Saving — tabs are restored at startup

### 🔨 In development / Needs revision

- 🔐 **Password Manager** — ❌ is not stable yet
- 📜 **Session history** — ❌ revision required
